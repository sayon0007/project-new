Links & Media Example

Links & Media Tags:

1. a tag → অন্য ওয়েবপেজ বা লিংকে যাওয়ার জন্য।
2. img tag → ওয়েবপেজে ছবি দেখানোর জন্য ।
3. audio tag → অডিও/গান বাজানোর জন্য ব্যবহার হয় (controls দিলে প্লে/পজ আসে)।
4. video tag → ভিডিও চালানোর জন্য ব্যবহার হয় (controls দিলে প্লে/পজ আসে)।
5. source tag → audio বা video এর ভেতরে মিডিয়া ফাইলের সোর্স দিতে ব্যবহার হয়।
6. track tag → ভিডিও/অডিওতে সাবটাইটেল বা ক্যাপশন যোগ করার জন্য ব্যবহার হয়।
7. iframe tag → অন্য ওয়েবপেজ বা ভিডিও (যেমন YouTube) এমবেড করার জন্য ব্যবহার হয়।

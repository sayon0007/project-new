নিচে HTML এ ব্যবহৃত লিস্ট ট্যাগসমূহের উদাহরণ দেয়া হলো:

1. Unordered List (বুলেট দিয়ে)

  • ul → Unordered list (বুলেট/ডট দিয়ে লিস্ট তৈরি করতে ব্যবহার হয়)।
  • ol → Ordered list (সংখ্যা/অক্ষর দিয়ে লিস্ট তৈরি করতে ব্যবহার হয়)।
  • li → ul বা ol এর ভেতরে প্রতিটি লিস্ট আইটেম লিখতে ব্যবহার হয়।
  • dl → Description list তৈরি করতে ব্যবহার হয়।
  • dt → Description list এ Term (শব্দ/শিরোনাম) লিখতে ব্যবহার হয়।
  • dd → Term এর ব্যাখ্যা/Description লিখতে ব্যবহার হয়।

4. Description List

নীচে একটি Description List এর উদাহরণ দেয়া হলো:

HTML
HyperText Markup Language – ওয়েবপেজের গঠন তৈরি করতে ব্যবহৃত হয়।
CSS
Cascading Style Sheets – ওয়েবপেজকে সুন্দর স্টাইল/ডিজাইন করতে ব্যবহৃত হয়।
JS
JavaScript – প্রোগ্রামিং ভাষা যা ওয়েবপেজকে ডায়নামিক ও ইন্টারেকটিভ করে।
Python
একটি জনপ্রিয় প্রোগ্রামিং ভাষা যা ওয়েব, ডেটা সায়েন্স, AI সহ অনেক কিছুর জন্য ব্যবহৃত হয়।
Java
একটি বহুল ব্যবহৃত প্রোগ্রামিং ভাষা যা মোবাইল অ্যাপ, ওয়েব অ্যাপ, এন্টারপ্রাইজ সফটওয়্যার ইত্যাদির জন্য ব্যবহৃত হয়।
C++
একটি শক্তিশালী প্রোগ্রামিং ভাষা যা সিস্টেম সফটওয়্যার, গেম ডেভেলপমেন্ট, এম্বেডেড সিস্টেম ইত্যাদির জন্য ব্যবহৃত হয়।

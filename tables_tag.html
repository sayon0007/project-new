নিচে HTML এ ব্যবহৃত Table Tag গুলোর কাজ ও ব্যবহার দেয়া হলো:

Table Tags (কাজ + ব্যবহার)

  1. table → টেবিল শুরু/শেষ করার জন্য।
  2. tr → টেবিলের একটি Row (সারি) তৈরি করতে ব্যবহার হয়।
  3. td → Row এর Data Cell (সাধারণ টেক্সট বা সংখ্যা) লিখতে।
  4. th → Table Header Cell (সিরির শিরোনাম) লিখতে।
  5. thead → টেবিলের Header অংশ আলাদা করতে।
  6. tbody → টেবিলের Main Body অংশ আলাদা করতে।
  7. tfoot → টেবিলের Footer অংশ আলাদা করতে।
  8. caption → টেবিলের Caption/Title লিখতে।
  9. colgroup / col → Column এর formatting বা style নির্ধারণের জন্য।

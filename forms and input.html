Forms & Input Tags – কাজ/ব্যবহার

HTML ফর্ম এবং ইনপুট ট্যাগ ব্যবহার করে ইউজার থেকে ডাটা সংগ্রহ করা হয়। ফর্মের মাধ্যমে ইউজার বিভিন্ন ধরনের ইনপুট দিতে পারে যেমন: টেক্সট, ইমেইল, পাসওয়ার্ড, চেকবক্স, রেডিও বাটন, ফাইল আপলোড ইত্যাদি। ফর্ম ডাটা সাধারণত সার্ভারে সাবমিট করা হয় প্রোসেসিং এর জন্য (যেমন: লগইন, রেজিস্ট্রেশন, সার্ভে ইত্যাদি)।

নিচে HTML ফর্ম এবং ইনপুট ট্যাগের বিভিন্ন ধরনের ব্যবহার দেখানো হয়েছে:

  1. form → এর কাজ হলো, সব ধরনের Tag (input field, button, select, textarea ইত্যাদি )একসাথে একটা container-এর ভিতরে রাখা।

  2. input → বিভিন্ন ধরনের user input field (text, email, password, checkbox, radio, file ইত্যাদি)।

  3. label → Input field এর নাম/বর্ণনা দিতে (accessibility এর জন্য গুরুত্বপূর্ণ)।

  4. textarea → Multi-line input field (যেমন comment box)।

  5. select → Dropdown menu তৈরি করতে।

  6. option → Dropdown এর একটি option/item।

  7. optgroup → Dropdown option গুলোকে group আকারে সাজাতে।

  8. fieldset → Form fields গুলোকে group করার জন্য একটি box।

  9. legend → Fieldset এর শিরোনাম/টাইটেল।

  10. datalist → Input field এর জন্য প্রি-ডিফাইনড suggestion list।

  11. button → Clickable button (submit/reset/normal)। ফর্ম সাবমিট করার জন্য সাবমিট বাটন ব্যবহার করা হয় এবং
    রিসেট বাটন দিয়ে ফর্মের সব ফিল্ড ক্লিয়ার করা যায়।

  12. output → কোনো ক্যালকুলেশন/ফলাফল দেখানোর জন্য।


Form Tag এর Attributes-গুলোর কাজ/ব্যবহারঃ

  1. action → ডাটা কোন URL/Server এ যাবে তা নির্ধারণ করে।

  2. method → ডাটা পাঠানোর পদ্ধতি (get = URL এ দেখাবে, post = লুকানো থাকবে)।

  3. enctype → ফাইল বা special ডাটা পাঠানোর সময় কিভাবে encode হবে (যেমন multipart/form-data)।

  4. autocomplete → পূর্বে দেওয়া তথ্য ব্রাউজার suggest করবে কি না (on / off)।

  5. novalidate → Form submit করার সময় HTML5 validation বন্ধ করবে।

  6. target → ফর্ম submit হলে কোথায় open হবে (_self, _blank, _parent, _top)।

  7. name → ফর্মকে একটি নাম দেওয়া হয়(JavaScript/Server side থেকে ব্যবহারের জন্য)।

  8. accept-charset → ফর্ম কোন character encoding ব্যবহার করবে (যেমন UTF-8)।


Input Tag এর Attributes-গুলোর কাজ/ব্যবহারঃ

  1. type → Input field এর ধরন (text, email, password, number, date, url, tel, color, range, checkbox, radio, file ইত্যাদি)।

  2. id → Input field এর ইউনিক আইডি (label এর for attribute এর সাথে লিঙ্ক করার জন্য)। ID মান এবং label এর for মান একই হতে হবে।

  3. name → Input field এর নাম (form সাবমিট করার সময় ডাটার key হিসেবে ব্যবহৃত হয়)। ইনপুটের নাম (server এ ডাটাগুলো কোন নাম/গ্রুপে যুক্ত হবে )।

  4. value → Input field এর ডিফল্ট মান/ভ্যালু। বাটনের ক্ষেত্রে বাটনের লেবেল হিসেবে কাজ করে। যা ইচ্ছা মত পরিবর্তন বা সেট করা যায়। value হলো আসল ডাটা যা backend এ যায়।

  5. placeholder → Input field এ হিন্ট/নির্দেশনা হিসেবে প্রদর্শিত টেক্সট।

  6. required → ইউজারকে ফিল্ডটি অবশ্যই পূরণ করতে হবে (ফাঁকা থাকলে সাবমিট হবে না)।

  7. readonly → ইনপুট ফিল্ডটি শুধুমাত্র পড়ার জন্য (ইউজার এটি সম্পাদনা করতে পারবে না)।

  8. disabled → ইনপুট ফিল্ডটিকে নিষ্ক্রিয় করে (ইউজার এটি ব্যবহার বা সম্পাদনা করতে পারবে না)। ফিল্ড নিষ্ক্রিয় (edit বা submit করতে পারবে না)।

  9. maxlength/minlength → ইনপুটে সর্বোচ্চ অক্ষরের পরিমাণ/সংখ্যা নির্ধারণ করে। কয়টা অক্ষর ইনপুট নিতে পারবে তা সীমাবদ্ধ করে।

  10. min / max → সংখ্যার ক্ষেত্রে সর্বনিম্ন এবং সর্বোচ্চ মান নির্ধারণ করে (number, date, range ইত্যাদি টাইপের জন্য)।

  11. step → সংখ্যার ক্ষেত্রে ইনপুটের ধাপ নির্ধারণ করে (যেমন 1,5,10,20 ইত্যাদি)। input value কত ধাপে বাড়বে/কমবে।

  12. checked → checkbox বা radio আগে থেকেই সিলেক্ট করা থাকে।

  13. multiple → ইনপুটে একাধিক ফাইল আপলোড করার অনুমতি দেয়।

  14. size → ইনপুট ফিল্ডের ভিজ্যুয়াল প্রস্থ নির্ধারণ করে (অক্ষরের সংখ্যা হিসেবে)।

  15. pattern → ইনপুটের জন্য একটি regex প্যাটার্ন নির্ধারণ করে (যেমন ইমেইল বা ফোন নম্বর ভ্যালিডেশনের জন্য)।

  16. list → ইনপুট ফিল্ডের জন্য একটি datalist এর সাথে লিঙ্ক করতে ব্যবহৃত হয়। datalist এর সাথে যুক্ত হয় (suggestion list দেয়)।

  17. accept → ফাইল ইনপুটের জন্য অনুমোদিত ফাইল টাইপ নির্ধারণ করে (যেমন image/*, .pdf ইত্যাদি)।

  18. autocomplete → ইউজারকে ইনপুট ফিল্ডের জন্য পূর্বে প্রবেশ করা মানগুলি সাজেস্ট করতে বলে। ব্রাউজার পুরনো ইনপুট suggest করবে কি না (on / off)।

  19. autofocus → পেজ লোড হওয়ার সাথে সাথে কার্সর এই ফিল্ডে থাকবে।

  20. tittle → ইনপুট ফিল্ডের উপর মাউস হোভার করলে টুলটিপ হিসেবে প্রদর্শিত হবে।


HTML input tag-এর type attribute-এ বিভিন্ন value দেওয়া যায়, যার মাধ্যমে field-এর ধরন নির্ধারণ হয়।

নিচে গুরুত্বপূর্ণ input type গুলোর তালিকা দেওয়া হলো:

  • text → সাধারণ টেক্সট ইনপুট ফিল্ড।
  • email → ইমেইল ঠিকানা ইনপুটের জন্য। (ভ্যালিডেশন সহ)
  • password → পাসওয়ার্ড ইনপুটের জন্য (টেক্সট লুকানো থাকে)।
  • number → শুধুমাত্র সংখ্যার ইনপুটের জন্য।
  • date → ক্যালেন্ডার থেকে তারিখ নির্বাচন করার জন্য।
  • url → ওয়েবসাইট URL ইনপুটের জন্য। (ভ্যালিডেশন সহ)
  • tel → ফোন নম্বর ইনপুটের জন্য।
  • color → রঙ নির্বাচন করার জন্য কালার পিকার।
  • range → একটি স্লাইডার দিয়ে একটি মান নির্বাচন করার জন্য।
  • checkbox → একাধিক অপশন থেকে এক বা একাধিক অপশন সিলেক্ট করার জন্য।
  • radio → একাধিক অপশন থেকে শুধুমাত্র একটি অপশন সিলেক্ট করার জন্য।
  • file → ফাইল আপলোড করার জন্য।
  • hidden → ইউজারের কাছে দৃশ্যমান নয় এমন ডাটা স্টোর করার জন্য।
  • submit → ফর্ম সাবমিট করার বাটন।
  • reset → ফর্মের সব ফিল্ড রিসেট করার বাটন।
  • button → সাধারণ বাটন (যা JavaScript দিয়ে কাস্টম কাজ করতে পারে)।
